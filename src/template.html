Proyectos 2021

valmovilidad.com

Página web que ofrece servicios de transportación en el Aeropuerto de Cancún. El cliente puede crear, ver y pagar su reservación de transporte para una fecha determinada. Se utilizó PHP, Yii2 Framework, JQuery, SCSS, y Gulp.js. Trabajé en la programación del proyecto completo, Frontend y Backend.

Point Of Sale (POS), para Etransfers

Página web orientada a una aplicación POS. Los usuarios de esta app, son vendedores de transportación en el Aeropuerto de Cancún, que utilizan una Tablet como dispositivo en sus puntos de venta. Se utilizó PHP, Yii2 Framework, JQuery, SCSS, y Gulp.js. Programé toda la parte Frontend del proyecto.

City Car Rental Puerto Vallarta
(puertovallartacarrental.net)

Página web dedicada a la Renta de autos en Puerto Vallarta. Trabajé en el rediseño de esta página, maquetando todo el sitio desde cero, utilizando mejores prácticas de HTML, CSS, y JS, con la finalidad de que el sitio se vea mejor, que tenga un aspecto de página web más actual, que mejore la velocidad de carga, y que tenga el código mejor ordenado.

airportcancun.com

Página web que brinda distintos servicios del Aeropuerto de Cancún. Servicios como por ejemplo: Transportación, renta de autos, tours y hoteles. Además, también cuenta con: Información acerca de Cancún, Playa del Carmen, Tulum, Puerto Aventuras, Playa Mujeres y Akumal; blog, monitoreo de vuelo, objetos perdidos, duty free, entre otros. Mi función fue maquetar absolutamente todo el sitio, teniendo en cuenta la experiencia de usuario, y conectar servicios desde el Frontend para el área de transportación.

See Live

Office Lunch Platform

Panel de control encargado de gestionar un sistema de comidas para empleados de mi actual trabajo. Los administradores pueden añadir comidas del día, y los usuarios pueder escoger su respectiva comida. Llevando así, un control de usuarios, comidas, y precios.
Este sistema es un CRUD, que desarrollé por mero hobby, ya que antes de este sistema, se utilizaba un excel, y no era muy práctico. En este proyecto cree mi propio Framework MVC en PHP.
